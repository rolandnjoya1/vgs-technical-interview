OVERVIEW 

MY Technical Documentation Guide. (Link)

The overview of this proof of concept(PoC) demonstrates how to implement VGS solutions into
payment processing applications that intergrate 3rd party payment providers such as stripe.

The POC will demonstrate how an inbound route can be configured to create a token for a customer VGS Vault
the outbound route will be configured to pass the token onto the third party stripe API which
will process the payment. 

***PREREQUISITES***

VGS Sandbox account. 
Strip API key.
AWS Account & Credentials.
